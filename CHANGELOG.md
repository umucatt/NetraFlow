# CHANGELOG

## 0.9.2

### 闪记精简
- 闪记流程收敛为“选择 → 输入 → 确认 → 完成写入”，实际页面保留选择、输入、确认。
- 选择页承担账户、输入模式和日期选择。
- 输入页保留顺序输入和键盘快捷键，移除可见输入框。
- 确认页兼具修改功能，移除旧修正页、暂存区、选区整体移动和选取工具。
- 写入后直接跳转目标账户详情页。
- 优化周视图日期显示、金额对齐、当前格、下一顺位格和未输入格状态。
- 新增可关闭的弱化快捷键提示。

### 全局搜索重构
- 全局搜索改为统一结果流，账户、历史记录、快照、设置项可混合排序。
- 移除自动分类切换，默认显示“全部”。
- 新增“允许推断”搜索设置，默认开启。
- 重做命中 / 推断规则，删除强相关、中相关、弱相关等旧描述。
- 优化数字、日期、金额搜索规则，日期型数字优先作为日期解析。
- 收紧金额近似推断，避免宽泛数字误匹配。
- 搜索高亮改为仅命中结果显示，推断结果不高亮。
- 右侧搜索预览补全账户、历史记录和快照信息。
- 历史记录搜索结果跳转至对应账户详情中的具体记录。
- 搜索结果展示根据命中字段显示净变动或余额变化。

### 视觉与交互一致性
- 统一确认框遮罩和禁用斜线覆盖样式。
- 优化历史记录日历、浏览记录筛选按钮和未来日期处理。
- 记一笔账户选择界面重新整理显示层级。
- 全 App 默认禁用非输入区域文字选取，输入框和可复制内容保留选取能力。
- 扩展大窗口空白区域返回逻辑。

### 关于页面与信息入口
- “联系我”调整为“获取信息”。
- 新增 GitHub Releases 链接。
- 删除“碎碎念”说明内容，并调整彩蛋区域层级。
- 当前版本更新为 0.9.2。
