# CHANGELOG


## 0.9.7

### 新增

* 新增 Windows 验证工作流，用于执行类型检查、测试和生产构建。
* 新增按需触发的 Windows 草稿发布工作流，用于生成安装版和便携版产物。
* 新增只读发布准备检查、发布产物校验和 SHA-256 校验和生成。
* 主存储文档新增 schema 1 格式。
* 新增经过校验的 previous 存储副本，用于在当前存储损坏时恢复。

### 调整

* AppData 中的分组、账户和历史记录改为一次批量事务提交。
* 主存储写入改为使用临时文件、fsync、读回校验、原子替换和最终校验。
* 测试运行器改为发现、转译、校验并执行完整 Node 测试套件，并正确传递失败状态。
* 开发环境 runtime 和 userdata 与已安装应用的数据隔离。
* 开发启动固定使用 Vite 5174 端口，并只在确认当前 Vite 进程就绪后启动 Electron。

### 修复

* 避免序列化或历史保护失败时产生部分 AppData 保存。
* 避免损坏的 storage 被当作首次启动空状态处理。
* 避免 future schema 被旧版本应用覆盖。
* final verify 或启动读取失败时，可从 previous 校验副本恢复。
* 避免 Electron 开发启动连接到占用 5174 端口的无关服务。

### 说明

* 现有有效的 0.9.6 flat storage 只读时保持不变，并在第一次真实保存时升级为 schema 1。
* future schema 会被安全拒绝，不会修改对应文件。
* 此版本未启用 Windows 代码签名和自动应用更新。


## 0.9.6

### 数据与快照

* 修复快照导入恢复逻辑，避免重复导入时叠加出多余记录。
* 新增快照导入记录，可查看导入时间、快照时间和导入前后的数据变化。
* 优化加密快照导出流程，启用加密时会先验证密码，再选择导出位置。
* 优化自动快照状态提示，可更直观看到上次备份、下次备份和执行状态。

### 账户

* 增加账户名称和账户类型名称的重复检查，归档账户也会参与名称占用判断。
* 重名时会在输入框中给出更轻量的提示，减少确认前后的反复操作。

### 图表与统计

* 优化折线图点值标签的位置，减少数字贴线或压线的情况。
* 优化动态纵轴范围，趋势变化更容易观察。
* 调整图表设置中的纵轴范围命名，使选项含义更直观。

### 记一笔

* 记一笔账户选择中，过长账户名会在省略时显示完整名称提示。

### 界面与交互

* 优化全局提示的位置与动效，减少对主要内容的遮挡。
* 统一部分弹窗遮罩表现，弹出和返回时的观感更加稳定。
* 调整部分禁用状态、设置页和图表页面中的视觉细节。
* 修复若干界面回归问题，提升多处操作流程的一致性。


## 0.9.5

### 界面与交互

* 窗口与主要界面的视觉进一步收口，层级、留白和操作状态更加克制。
* 新增“页面重心”设置，可在双栏页面中调整主要内容更偏左或偏右显示。
* 精简实体页面返回按钮，更多使用 Esc、空白区域点击和统一的全局返回方式。
* 统一自绘说明框和提示信息，减少重复说明与不必要打扰。

### 主题与启动

* 优化冷启动主题同步，减少浅色和深色主题切换时的闪屏感。
* 调整窗口框架与启动阶段的显示关系，使进入主界面前后的观感更稳定。

### 闪记

* 优化闪记选择、输入和确认流程，移除多余返回或退出控件，记录过程更专注。
* 修复闪记流程中 Esc 阶段返回不符合预期的问题。

### 账户、搜索与双栏页面

* 修复账户类型双击进入图表时会先展开分组的问题。
* 改善搜索、历史、快照、账户新增和账户恢复等双栏页面在页面重心切换后的显示关系。

### 安装与维护

* 优化安装版卸载清理，runtime 等运行期残留会被清理，个人数据仍按卸载时的选择处理。
* 整理界面底层结构，提高后续维护稳定性。


## 0.9.4

### 数据与快照

* 快照、加密快照和用户配置文件新增完整性校验，导入可能被修改过的文件时会提示风险。
* 优化快照和用户配置文件的导出格式，减少文件体积。
* 优化本地文件写入流程，降低异常中断导致文件损坏的风险。
* 优化加密快照导入导出流程，密码输入、错误提示和完整性校验体验更加统一。

### 历史记录

* 历史记录日历新增记录密度显示，便于快速判断不同日期的记录数量差异。
* 优化历史记录页面的列表布局、日历筛选、日期输入和未来日期处理，筛选范围更容易理解。
* 优化账户详情中的历史记录折叠显示，同日多条记录的摘要、排序和展开结果更加清晰。
* 优化历史记录条目展示，备注、来源标记、折叠组和金额变化的显示更加稳定。
* 修复历史记录折叠组外层错误显示闪记或汇总导入来源标记的问题。
* 修复历史记录日期范围输入中反向日期、非相邻月份和未来日期处理不一致的问题。
* 修复账户详情历史记录中同日多条记录摘要和展开顺序不一致的问题。

### 闪记

* 优化闪记日期选择方式，默认使用拖选日期，单日和连续日期选择更直观。
* 优化闪记输入与确认流程，减少不必要的返回操作和干扰提示。
* 优化闪记写入后的跳转和历史记录显示，使记录完成后的查看流程更连贯。

### 账户

* 优化账户操作流程，账户编辑、金额修改、归档、恢复和删除操作更加稳定。
* 优化账户自定义缩写输入，不再限制输入框输入长度，改善中文、日文等输入法体验；实际显示仍保留最多四个有效字符。
* 优化删除账户后的数据处理，已删除账户及其相关历史记录不再参与趋势、占比和图例显示。
* 修复自定义缩写输入时，输入法组合文本可能被长度限制打断的问题。
* 修复删除账户后，部分图表中可能出现“已删除旧账户”占位项的问题。

### 图表与统计

* 优化首页、资产总览和图表展示，统计信息、图表空状态、占比图和图例显示更加协调。
* 优化账户类型详情图表，账户占比图与总资产占比图的视觉风格更加一致。
* 移除账户类型详情图表中无实际作用的“类型属性”展示。

### 搜索与设置

* 优化全局搜索和设置项定位体验，设置相关结果更容易判断和跳转。
* 优化设置页面布局，显示与界面、图表、全局搜索、数据与备份、安全等设置区域更加统一。
* 修复全局搜索设置页中控件位置与其他设置页不一致的问题。

### 汇总导入

* 优化汇总导入页面的提示词、导入区、预览区和账户分配流程显示。
* 保持原有校验、风险提示、账户匹配、全部导入和全部舍弃规则，导入前核对流程更加稳定。

### 示例模式与重置

* 优化示例模式进入、退出和真实数据恢复流程。
* 优化重置流程，确认和恢复状态更加稳定。

### 整体体验

* 整理多个核心页面和操作流程，提升账户、历史记录、快照、搜索、设置、闪记、汇总导入和图表等功能的稳定性与一致性。

## 0.9.2

### 闪记
- 精简闪记流程，选择、输入、确认和写入后的跳转更加连贯。
- 优化闪记输入与确认体验，日期、金额、当前格和未输入状态显示更清晰。
- 弱化快捷键提示，让闪记页面更专注于记录本身。

### 全局搜索
- 优化全局搜索结果排序和匹配体验，账户、历史记录、快照和设置可以混合搜索。
- 支持更自然的数字、日期和金额搜索，减少宽泛数字带来的误匹配。
- 搜索高亮更聚焦于实际命中的内容，推断结果不再过度高亮。
- 搜索结果预览信息更完整，账户、历史记录和快照结果更容易判断。

- 调整汇总导入风险提示，不再因重复导入或 change 与 balance 混用直接阻断。

- 简化快照页面进入流程，减少进入前的额外步骤。
- 手动快照的上次备份时间改为相对显示，更容易快速理解。
- 快照相关操作、提示和禁用状态显示更统一。

### 界面与交互
- 记一笔与闪记的账户选择视觉进一步收口，层级更稳定。
- 图表、设置、历史、快照和弹窗等界面视觉一致性优化。
- 页面圆角、按钮、输入框、禁用状态和遮罩显示更统一。
- 历史记录浏览、日历筛选和未来日期处理更清晰。
- 非输入区域默认减少误选文字，输入框和可复制内容仍保留选择能力。
- 空白区域返回、弹窗返回和确认交互更加一致。
