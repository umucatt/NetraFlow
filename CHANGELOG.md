# CHANGELOG

## 0.9.3

### 新增
- 新增示例模式入口跳转，便于从示例数据回到正常使用流程。

### 改进
- 优化安装与卸载流程，安装版使用当前用户安装并调整默认安装路径选择。
- 调整安装版与便携版打包流程，发布产物按版本目录输出。
- 用户数据与运行缓存分离，账户数据和运行时缓存边界更清晰。
- 账户数据模型改为类别与账户分离，账户类别删除和归档账户恢复目标选择更明确。
- 图表数据派生逻辑模块化，提升图表数据维护一致性。
- 改进闪记和搜索体验，减少记录与定位时的干扰。

### 修复
- 修复历史记录时间显示问题。
- 修复安装版和便携版图标与文件版本资源显示。

### 技术调整
- 收口本地构建、安装版打包、便携版打包和 release 清理命令。

## 0.9.2

### 闪记
- 精简闪记流程，选择、输入、确认和写入后的跳转更加连贯。
- 优化闪记输入与确认体验，日期、金额、当前格和未输入状态显示更清晰。
- 弱化快捷键提示，让闪记页面更专注于记录本身。

### 全局搜索
- 优化全局搜索结果排序和匹配体验，账户、历史记录、快照和设置可以混合搜索。
- 支持更自然的数字、日期和金额搜索，减少宽泛数字带来的误匹配。
- 搜索高亮更聚焦于实际命中的内容，推断结果不再过度高亮。
- 搜索结果预览信息更完整，账户、历史记录和快照结果更容易判断。
- 历史记录搜索结果可以跳转到对应账户详情中的具体记录。
- 金额类结果会根据命中内容展示变动金额或余额变化。

### 汇总导入
- 调整汇总导入风险提示，不再因重复导入或 change 与 balance 混用直接阻断。
- 导入前的记录分组、账户分配和风险摘要更便于核对。

### 快照
- 简化快照页面进入流程，减少进入前的额外步骤。
- 手动快照的上次备份时间改为相对显示，更容易快速理解。
- 快照相关操作、提示和禁用状态显示更统一。

### 界面与交互
- 记一笔与闪记的账户选择视觉进一步收口，层级更稳定。
- 图表、设置、历史、快照和弹窗等界面视觉一致性优化。
- 页面圆角、按钮、输入框、禁用状态和遮罩显示更统一。
- 历史记录浏览、日历筛选和未来日期处理更清晰。
- 非输入区域默认减少误选文字，输入框和可复制内容仍保留选择能力。
- 空白区域返回、弹窗返回和确认交互更加一致。

### 关于页面
- 关于页“获取信息”入口调整。
- 新增 GitHub Releases 链接入口。
- 当前版本更新为 0.9.2。
